# V1 页面规格

日期：2026-06-06

## 1. 页面总览

V1 页面只围绕故事类短视频生产主线设计。

页面列表：

1. 工作台首页。
2. 开始创作向导。
3. 生成进度页。
4. 结构化结果编辑页。
5. 发布前质检页。
6. 导出内容包页。
7. 发布记录页。
8. 简单复盘页。
9. 管理员模型设置页。

## 2. 工作台首页

主按钮：

- 开始创作。

次入口：

- 最近项目。
- 发布复盘。
- 设置。

首页不得默认展示：

- 模型配置。
- 成本统计。
- 模板市场。
- 复杂数据看板。
- 自动发布。

验收：

- 新用户第一眼知道应该点击“开始创作”。
- 首页只有一个视觉上最突出的主按钮。

## 3. 开始创作向导

必填字段：

- 创作目标。
- 目标平台。
- 题材。
- 一句话想法。

可折叠高级字段：

- 目标受众。
- 语气风格。
- 视频时长。
- 质量/成本模式。

主按钮：

- 生成素材包。

验收：

- 用户无需填写 Prompt。
- 必填项不超过 4 个。

## 4. 生成进度页

生成步骤：

1. 理解想法。
2. 生成开头钩子。
3. 生成脚本。
4. 生成分镜。
5. 生成字幕。
6. 生成标题。
7. 生成封面文案。
8. 生成发布文案。
9. 运行质检。

失败操作：

- 重试当前步骤。
- 使用备用模型。
- 保存已完成结果。

验收：

- 失败时不会丢失已经生成的内容。
- 错误信息用用户能理解的语言展示。

## 5. 结构化结果编辑页

内容区块：

- 开头钩子。
- 脚本。
- 分镜。
- 字幕。
- 标题选项。
- 封面文案。
- 发布文案。
- 质检摘要。

编辑按钮：

- 开头更吸引人。
- 改得更短。
- 改得更爽。
- 改得更悬疑。
- 改得更口语。
- 换一组标题。
- 换一个封面文案。
- 重写结尾。
- 降低 AI 味。
- 更适合当前平台。

主按钮：

- 运行发布前检查。

验收：

- 结果不是一整段文本。
- 用户能局部修改，不需要重新生成全部内容。

## 6. 发布前质检页

检查项：

- 钩子强度。
- 情绪密度。
- 冲突清晰度。
- 信息增量。
- 口语化程度。
- 画面可执行性。
- 平台适配度。
- 同质化风险。
- 版权/敏感风险。
- AI 痕迹风险。

主操作：

- 一键修复问题。
- 继续编辑。
- 导出内容包。

验收：

- 评分低于阈值时，不默认推荐发布。
- 每个低分项都有具体修改建议。

## 7. 导出内容包页

默认导出：

- Markdown 内容包。
- 纯文本字幕。
- 分镜表。
- 发布文案。

导出包必须包含：

- 标题。
- 封面文案。
- 开头钩子。
- 脚本。
- 分镜。
- 字幕。
- 发布文案。
- AI 内容标识提醒。

主按钮：

- 导出。

下一步提示：

- 去平台手工发布。
- 发布后回来记录数据。

## 8. 发布记录页

字段：

- 平台。
- 发布时间。
- 内容链接。
- 播放量。
- 点赞。
- 评论。
- 收藏。
- 涨粉。
- 备注。

主按钮：

- 保存并查看复盘。

## 9. 简单复盘页

V1 只展示：

- 最近内容表现。
- 表现最好的题材。
- 表现最好的钩子类型。
- 表现最好的标题结构。
- 下一条内容建议。

不做完整 BI。

## 10. 管理员模型设置页

字段：

- 供应商。
- Base URL。
- API Key。
- 默认模型。
- 连接测试。
- 最近测试结果。

验收：

- 普通创作用户不需要进入此页面。
- 连接失败时展示可理解错误。
